Usage
	python main.py -dn <str> <-rs <int>> <-nb>

Args:
	-dn <str>
	Required, specifies the name of the data. Please use:
		breast-cancer
		glass
		iris
		soybean
		vote

	-rs <int>
	Optional, specifies the random_state of the data. Defaults to 1

	-nb
	Optional, specifies to run Naive Bayes. Defaults to Winnow2
